CF7 #14462 — Site-Wide Context Loss

The primary inline CTA form on 24+ landing pages captures name, email, and phone — but no page source, no transaction type, and no lead intent. Every submission arrives in HubSpot with zero context.

Date: 2 April 2026 CF7 Form ID: 14462 Severity: High — affects all landing pages Verified by: Playwright scan of 24 pages

CF7 #14462 is the most widely deployed form on the site — and it captures no context

This form appears 3 times on every landing page (hero section, mid-page CTA, footer CTA). When a user submits from /first-home-buyer-loans/ versus /refinance-home-loan/ versus /home-loans-for-doctors/, HubSpot receives the same context-free submission: name, email, phone. Nothing else.

The team cannot tell:

— Which page the lead came from
— What type of loan they're interested in
— Whether they're a first home buyer, investor, refinancer, or professional program lead
— Which CTA on the page they clicked (hero vs mid-page vs footer)

6. The Fix

Step 1: Add hidden fields to CF7 #14462

In the WordPress CF7 form editor for form ID 14462, add:

[hidden source_url default:shortcode_attr class:source-url]
[hidden transaction_type default:shortcode_attr class:transaction-type]

Step 2: Populate hidden fields via JavaScript

Add a script to the theme (or via a WP code snippet plugin) that runs on page load:

document.addEventListener('DOMContentLoaded', function() {
  // Populate source_url with the current page path
  document.querySelectorAll('form.wpcf7-form input[name="source_url"]').forEach(function(input) {
    input.value = window.location.pathname;
  });

  // Infer transaction_type from the page URL
  var path = window.location.pathname;
  var transactionType = 'general';

  if (path.includes('first-home') || path.includes('kiwi-first')) {
    transactionType = 'first_home_buyer';
  } else if (path.includes('refinanc') || path.includes('lower-your-rate') || path.includes('upgrade')) {
    transactionType = 'refinance';
  } else if (path.includes('invest') || path.includes('buy-to-let')) {
    transactionType = 'investment';
  } else if (path.includes('doctor') || path.includes('nurse') || path.includes('teacher')
             || path.includes('pharmacist') || path.includes('midwi')) {
    transactionType = 'professional_program';
  } else if (path.includes('guarantor')) {
    transactionType = 'guarantor';
  } else if (path.includes('commercial') || path.includes('business')) {
    transactionType = 'commercial';
  } else if (path.includes('apartment')) {
    transactionType = 'apartment';
  } else if (path.includes('mortgage-broker-') || path.includes('home-loans-brisbane')) {
    transactionType = 'location_page';
  }

  document.querySelectorAll('form.wpcf7-form input[name="transaction_type"]').forEach(function(input) {
    input.value = transactionType;
  });
});
